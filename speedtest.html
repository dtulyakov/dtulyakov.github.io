Проверка скорости интернета

by Денис Туляков
in wiki on 2016-06-03 | tags: Speedtest CLI Python | comments

Если надо быстро проверить примерную скорость интернета, но всё происходит на сервере и нет второго куда можно постучать с помощью Iperf

В прочтименя на гитхабе автора есть примеры установки и запуска

1
2
3
pip install speedtest-cli
pip3 install speedtest-cli
easy_install speedtest-cli

1
2
pip install git+https://github.com/sivel/speedtest-cli.git
git clone https://github.com/sivel/speedtest-cli.git && python speedtest-cli/setup.py install

1
2
wget -O speedtest-cli https://raw.githubusercontent.com/sivel/speedtest-cli/master/speedtest_cli.py
chmod +x speedtest-cli

1
2
curl -Lo speedtest-cli https://raw.githubusercontent.com/sivel/speedtest-cli/master/speedtest_cli.py
chmod +x speedtest-cli

Можно одной строчкой скачать и сразу запустить

1
wget -O - https://raw.github.com/sivel/speedtest-cli/master/speedtest_cli.py | python
